# ABACADEMY.COM

ABACADEMY.COM is an online MDCAT preparation platform.

## Features

- Student Registration & Login
- Admin Dashboard
- Practice MCQs
- Online Tests
- Subject-wise Practice
- Chapter-wise Practice
- Topic-wise Practice
- Wrong Question Bank
- Bookmarked MCQs
- Online Rankings
- Result History
- Notes Store
- PDF Notes
- Notifications
- Analytics Dashboard
- One Device Login
- Excel MCQ Upload
- Profile Management
